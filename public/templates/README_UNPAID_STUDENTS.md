# Unpaid Students Template

## How to Use This Template

1. **Download the CSV file**: `unpaid_students_template.csv`
2. **Open in Excel**: Double-click the CSV file or open it with Microsoft Excel
3. **Fill in student information**: Add rows for all students who did not pay full tuition

## Column Definitions

| Column Name     | Description                                      | Example        |
|-----------------|--------------------------------------------------|----------------|
| Student Name    | Full name of the student                         | John Doe       |
| Class           | Student's grade/class                            | Grade 6        |
| Expected Fees   | Total tuition fees expected for the term (UGX)   | 1000000        |
| Amount Paid     | Amount actually paid by student (UGX)            | 600000         |
| Balance         | Outstanding balance (Expected - Paid) (UGX)      | 400000         |

## Important Notes

- **Do NOT change the column headers** (first row)
- **Use numbers only** for fees, amount paid, and balance (no commas or currency symbols)
- **Example**: Use `1000000` instead of `1,000,000` or `UGX 1,000,000`
- The balance will be automatically calculated in the preview (Expected Fees - Amount Paid)
- Save the file as `.xlsx` or `.xls` format before uploading

## Sample Data

The template includes 4 sample rows. You can:
- Delete the sample rows and add your own data
- Or replace the sample data with your actual student information

## File Formats Accepted

- Excel 2007+ (.xlsx) - **Recommended**
- Excel 97-2003 (.xls)
- CSV (.csv)

## Maximum File Size

- 5 MB
