Unpaid Students Template - Instructions

Purpose: Use this template to upload a list of students who did not pay full tuition fees for the last term.

How to Use This Template

  1. Download the CSV template file using the button below
  2. Open the file in Microsoft Excel, Google Sheets, or any spreadsheet application
  3. Fill in your student information following the column format
  4. Save the file as .xlsx or .xls format (Excel format)
  5. Upload the file in the assessment form

Column Definitions

Column Name Description Example
Student Name Full name of the student John Doe
Class Student's grade or class level Grade 6
Expected Fees Total tuition fees expected for the term (UGX) 1000000
Amount Paid Amount actually paid by the student (UGX) 600000
Balance Outstanding balance (Expected - Paid) in UGX 400000

Important Notes

  • Do NOT change the column headers (first row must remain: Student Name, Class, Expected Fees, Amount Paid, Balance)
  • Use numbers only for fees - no commas or currency symbols
  • Correct: 1000000 | Wrong: 1,000,000 or UGX 1,000,000
  • The balance will be automatically calculated during preview
  • Delete the sample rows before adding your data
